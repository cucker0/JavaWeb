## ServletContext类
### 什么是ServletContext
```text
ServletContext是一个接口
ServletContext是一个域对象
每个web工程，都对应一个ServletContext对象
```

### ServletContext作用
* 可以获取web.xml文件中的配置上下文参数
* 可以获取web工程在服务器上的工程名
* 可以获取web工程中文件病人或文件在服务器上的绝对路径
* 可以设置、获取web工程的全局属性
