## html中"\r\n"与"<br/>"的区别
```text
\n  是换行，英文是New line，表示使光标到行首
\r  是回车，英文是Carriage return，表示使光标下移一格
\r\n  表示回车换行，说白了，就是windows的换行符，一般也可以直接用\n替换

\\  反斜杠
\$  美圆符
\"  双引号
\'  单引号


\r\n是输出的HTML代码换行，客户看到的效果没有换行。

如果是输出给浏览器，就用<br/>

\r\n，好比你在idea里做一个网页，在源代码里按一下回车，是给源代码回车换行。
是输出一个（HTML标记性语言），在HTML中解释为输入的字换行。
\r\n的作用是给源代码回车换行，输入后，查看html源代码，就看会到有换行，
而HTML解释器（浏览器）认为它是没用的东西，用不用都一样显示。
```
